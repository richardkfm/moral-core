# Roadmap

**Moral Core Development Roadmap**

This roadmap describes planned development phases. Timelines are approximate and depend on community contributions.

---

## v0.1.0 -- Initial Release (Current)

- 16 ethical skill domains with full documentation
- Shared doctrine (PRINCIPLES.md)
- Safety and limitations documentation
- Evaluation framework with adversarial tests, scenarios, and rubrics
- Integration guides for 8 deployment contexts
- 7 pre-built policy bundles
- 5 Claude Code subagent definitions
- GitHub templates for contributions
- MIT license

**Status:** In progress

---

## v0.2.0 -- Community Feedback

- Incorporate feedback from early adopters
- Refine skill boundaries based on real-world deployment experience
- Add community-contributed test cases and adversarial prompts
- Improve edge case handling across all skills
- Begin localization framework for non-English deployments
- Add skill metadata manifests (YAML) for programmatic loading

---

## v0.3.0 -- Automated Evaluation Tooling

- Scripts for automated skill evaluation against test cases
- CI/CD pipeline for regression testing on skill changes
- Benchmark runner that scores skill effectiveness across scenarios
- Comparison tooling for evaluating skill performance across models
- Dashboard for tracking evaluation results over time

---

## v0.4.0 -- Integration SDK

- Python SDK for loading and composing skills programmatically
- TypeScript/JavaScript SDK
- Pre-built policy bundle loader
- Configuration validation tooling
- Integration examples for popular agent frameworks (LangChain, AutoGen, CrewAI, etc.)

---

## v1.0.0 -- Stable Release

- All skills reviewed by at least one domain expert
- Full test coverage across all skills
- Stable API for programmatic skill loading
- Comprehensive documentation with tutorials
- Published evaluation results across major model families
- Formal versioning commitment (semver for skill content)

---

## Future Directions

- **Multilingual support.** Translate skills and documentation into major world languages. Adapt cultural notes for each language community.
- **Domain-specific extensions.** Specialized skill packs for healthcare, legal, financial, and educational contexts.
- **Interactive training.** Tools for training teams on ethical skill integration.
- **Certification concept.** Explore a voluntary certification or self-assessment framework for systems using Moral Core skills.
- **Model-specific tuning.** Optimized skill variants for specific model families.
- **Research partnerships.** Collaborate with AI safety researchers to validate and improve skill effectiveness.
- **Robotics test scenarios.** Physical-world test protocols for robots using Moral Core skills.
